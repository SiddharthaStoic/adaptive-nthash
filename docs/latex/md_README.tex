nt\+Hash = nt\+Hash is a recursive hash function for hashing all possible k-\/mers in a DNA/\+RNA sequence.\hypertarget{md_README_autotoc_md0}{}\doxysection{\texorpdfstring{Installation \& Usage}{Installation \& Usage}}\label{md_README_autotoc_md0}
Make sure \href{https://mesonbuild.com/}{\texttt{ Meson}} is installed on the system.

Download the repo (either from the releases section or close using {\ttfamily git clone \href{https://github.com/bcgsc/ntHash}{\texttt{ https\+://github.\+com/bcgsc/nt\+Hash}}}). Setup meson in an arbitrary directory (e.\+g. {\ttfamily build}), by running the following command in the project\textquotesingle{}s root\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{meson\ setup\ build}

\end{DoxyCode}


Then, build the project and its dependencies using\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{cd\ build\ \&\&\ ninja}

\end{DoxyCode}


The project\textquotesingle{}s static library ({\ttfamily libnthash.\+a}) and tests ({\ttfamily nthash}) will be generated alongside in the {\ttfamily build} folder.

Tests can be run using {\ttfamily ninja} in the {\ttfamily build} directory\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{ninja\ test}

\end{DoxyCode}
\hypertarget{md_README_autotoc_md1}{}\doxysubsubsection{\texorpdfstring{Static Library Usage}{Static Library Usage}}\label{md_README_autotoc_md1}
To use nt\+Hash in a C++ project\+:
\begin{DoxyItemize}
\item Link the code with {\ttfamily libnthash.\+a} (i.\+e. pass {\ttfamily -\/Lpath/to/nthash/build -\/lnthash} to the compiler).
\item Add the {\ttfamily include} directory (pass {\ttfamily -\/I path/to/nthash/include} to the compiler).
\item Import nt\+Hash in the code using {\ttfamily \#include \texorpdfstring{$<$}{<}nthash/nthash.\+hpp\texorpdfstring{$>$}{>}}.
\item Access nt\+Hash classes from the {\ttfamily nthash} namespace.
\end{DoxyItemize}\hypertarget{md_README_autotoc_md2}{}\doxysection{\texorpdfstring{Examples}{Examples}}\label{md_README_autotoc_md2}
